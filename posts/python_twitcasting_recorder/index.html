使用python錄TwitCasting直播

post date - 2022/07/30
last update - 2022/08/06

使用python錄製https://twitcasting.tv/直播的方法

使用別人寫好的程式，點右上的code->donwload zip整包下載後解壓縮。
https://github.com/prinsss/twitcasting-recorder

安裝python環境

官方網站下載並安裝
https://www.python.org/
如果有修改安裝路徑請記下來，晚點設定時需要使用。

升級pip
直接在路徑輸入cmd叫出提示字元

使用cmd命令提示字元，輸入py -m pip install --upgrade pip

使用pip安裝相關套件需要先進入系統設定系統的path，打開設定->系統->進階系統設定->環境變數->找到系統變數中的Path查看

由於權限的問題安裝後有可能沒有這兩個參數。
如果python和Python39/script沒有在參數內的話，新增這兩個路徑
預設安裝路徑是C:\Users[使用者名稱]\AppData\Local\Programs\Python\Python39
如果沒更改的話新增
C:\Users[使用者名稱]\AppData\Local\Programs\Python\Python39\Scripts
C:\Users[使用者名稱]\AppData\Local\Programs\Python\Python39
一路按確定儲存

安裝錄影需要的套件

下載的資料夾內開啟cmd，執行pip install -r requirements.txt安裝相關套件，輸入後等待跑完就好

接下來就可以錄影了
一樣在資料夾內執行cmd

main.py [錄影的id]

或

main.py [錄影的id] -o [儲存的檔案名稱].ts

[錄影的id]是twitcasting.tv/後的文字
EX:如果要錄 https://twitcasting.tv/abcdef
則輸入main.py abcdef
錄影中如果需要中途停止則按下ctrl+c
